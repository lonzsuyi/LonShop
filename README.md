# Running the sample

####  Environmental requirements :

1. DotNet Core ( Version < 6.0.x )

2. NodeJS (Version < 15.0.x )

#### CLI :
<pre>
$ cd LonShop/src/LonShopWeb
$ dotnet run
</pre>

#### Front-end code in :
<pre>
$ cd LonShop/src/LonShopWeb/client_app
$ npm run start
</pre>
